設定と一括処理

縦に長くならないよう、設定を目的別の折りたたみセクションに整理しました。

データとモデル 読み込み / モデル / 背景
同一モデルのピーク和として 1〜6 個まで同時フィットできます。
例: 0.15 は選択範囲の左右それぞれ15%の点群で線形背景を推定します。
ピーク初期値 複数列入力 / 固定 / 共有ID
各定数は「固定」で最適化対象から外せます。「共有ID」に同じ文字列を入れると、複数ピークでその定数を共通化します。
一括処理と書き出し 連続フィット / ZIP / JSON / CSV
オンにすると、1個目の結果を2個目へ、2個目の結果を3個目へという形で初期値を順次引き継げます。
ZIP出力時に、背景差し引き後の y 軸だけを残す2列版か、元データと背景も含む4列版を選べます。.txt / .dat を選んだ場合は、区切り文字をカンマ・タブ・スペースから切り替えられます（.csv は常にカンマです）。
