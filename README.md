# Peak Fitting PWA

ブラウザだけで動く、単一ピーク/複数ピークに対応した簡易ピークフィッティングPWAです。

## 対応機能

- TXT / CSV / DAT の 2 列数値データ読み込み
- グラフ上ドラッグによるフィット範囲選択
- 線形背景の推定と差し引き
- Gaussian / Lorentzian / Voigt (pseudo-Voigt) / BWF
- 同一モデルの複数ピーク和フィット（1〜6ピーク）
- 複数ファイルに対する同一初期条件での一括フィット
- 複数ファイルの線形背景差し引き結果を ZIP で一括出力（ZIP内ファイルの拡張子・列構成・テキスト区切り文字を選択可能）
- 結果の CSV 出力
- オフライン利用用の Service Worker / Web App Manifest

## 使い方

1. `index.html` をローカルサーバー経由で開く
2. 複数のテキストファイルを読み込む
3. グラフ上をドラッグしてフィット範囲を指定する
4. モデルとピーク数を選ぶ
5. 必要なら「選択範囲から初期値を再推定」を押す
6. 単独フィット・一括フィット、または「背景差し引きのみ（ZIP）」を実行する
7. フィット結果は CSV、背景差し引き結果は ZIP として保存する（ZIP内データは .csv / .txt / .dat に加え、x と背景差し引き後 y の2列版 / 4列版、および .txt / .dat 用のカンマ / タブ / スペース区切りを選択可能）

## ローカル起動例

### Python

```bash
cd peakfit-pwa
python -m http.server 8000
```

### Node.js

```bash
cd peakfit-pwa
npx serve .
```

その後、ブラウザで `http://localhost:8000` を開いてください。

## 実装メモ

- Voigt は厳密畳み込みではなく pseudo-Voigt 近似です。
- フィットは同一モデルの 1〜6 ピーク和を想定しています。
- 最適化はブラウザ内の Nelder–Mead 法で行っています。
- 背景は「選択範囲の左右端部の点群」から線形回帰で推定します。
